SYSTEM REQS: python 2.x, google-api-python-client-1.6.2, BeautifulSoup4, requestsmodule

DESCRIPTION: goes to 92Q radio station's web site and scrapes playlist. goes to youtube and finds youtube URL for each song, then downloads with youtube-dl

[92Q is a Baltimore hip-hop radio station. in theory, the program could be modified to download playlists/songs from anywhere.]
